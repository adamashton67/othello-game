# Othello Game

This is a Python project to build a playable version of Othello.

The aim of the project is for me to practise building a structured software project using Python, Git, testing, and clear commits.

## Version 1 Goal

The first playable version will be a local two-player game with:

- An 8x8 board
- Black and white pieces
- Click-to-place moves
- Valid move checking
- Automatic piece flipping
- Turn indicator
- Live black and white piece totals
- Game-over detection
- Winner display

## Future Ideas

Possible future improvements include:

- Highlighting valid moves
- Restart button
- Main menu
- Simple AI opponent
- Difficulty levels
- Animations
- Sound effects
